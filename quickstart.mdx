## 1. Cloning the Repo
In your terminal run the following commands individually

```terminal terminal
git clone link [YOUR APP NAME]
```
```terminal terminal
cd [YOUR APP NAME]
```
```terminal terminal
npm install
```
```terminal terminal
git remote remove origin
```

## 2. Environment Variable Setup
In the route directory of the project, create a `.env` file and input the following
```terminal .env
BASE_URL=http://localhost:5173/
MAILJET_API_KEY=
MAILJET_SECRET_KEY=
STRIPE_SECRET_KEY=
TRELLO_QUEUE=
TRELLO_TOKEN=
```

## 3. Supabase
<Steps>
  <Step title="Start a project">
    Head over to <a href="https://supabase.com" target="_blank">Supabase</a> and create a new project
  </Step>
  <Step title="Get API credentials">
    Within your project dashboard, go to Project Settings and then click API
  </Step>
  <Step title="Paste into app">
    Copy and paste the project URL and the `anon` `public` API key into `hooks.ts`
    ```typescript src/hooks.ts
    // Supabase client
    const SUPABASE_URL = [YOUR URL]
    const SUPABASE_ANON_KEY = [YOUR API KEY]

    ```
    <Note>It is completely safe to expose your public Supabase API key, the <a href="features/database" target="_blank">Database</a> section covers this in more detail</Note>
  </Step>
</Steps>

## 4. Development Server (localhost)
To start the development server you can use the following command
```terminal terminal
npm run dev
```
<Warning>Without setting up Supabase, your app will not be able to start the development server</Warning>
